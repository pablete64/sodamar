# SODAMAR™ — Ingeniería Industrial Avanzada

Sitio web corporativo de **SODAMAR**, empresa especializada en automatización industrial, robótica, PLC/SCADA, optimización logística y soluciones para el sector agroalimentario.

## Tecnologías

- **React 18** + TypeScript
- **Vite** (build tool)
- **Tailwind CSS** + shadcn/ui
- **Framer Motion** (animaciones)

## Desarrollo local

```bash
npm install
npm run dev
```

El servidor arranca en `http://localhost:8080`.

## Build de producción

```bash
npm run build
npm run preview
```

## Estructura

```
src/
├── components/     # Secciones y componentes UI
├── pages/          # Páginas (Index, NotFound)
├── assets/         # Logo e imágenes
├── hooks/          # Custom hooks
├── lib/            # Utilidades
└── index.css       # Estilos globales y variables CSS
```
